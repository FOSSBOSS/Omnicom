The Goal of OMNICOM is to create a simple, universal PLC control interface. 

There is a lot of little bits and bobs to get that working the way I envission it. 
  While the communication structure is similar, the exact Registers in use, and instrructions do vary. 
  The idea is to create an array of Control and Debugging registers, per OEM, Unit, model.
  Then use this app as a means to interface with the other PLCs.
